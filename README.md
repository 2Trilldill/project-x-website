# Project X Construction LLC Website

This is a responsive business website built for **Project X Construction LLC**, a general contracting company based in North Carolina. The site was developed using **React** and styled with **custom CSS** (after initially prototyping in Tailwind CSS). It includes essential pages such as:

- Home
- About Us
- Services
- Projects
- Contact
- Inquiry form

The project highlights component-based UI architecture with **React Router**, a mobile-friendly navigation bar, and form handling ready for future backend integration. Designed to be professional, accessible, and easily extendable for future features such as photo galleries or customer testimonials.

---

## 🧰 Technologies Used

- React
- React Router
- HTML5 / CSS3
- Git & GitHub
- Visual Studio Code

---

## 🚀 How to Run Locally

1. Clone the repository  
   `git clone https://github.com/your-username/project-x-website.git`

2. Navigate to the project directory  
   `cd project-x-website`

3. Install dependencies  
   `npm install`

4. Start the development server  
   `npm run dev`
